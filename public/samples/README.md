# Sample Files

Use these files to test the browser converter:

- `sample.csv`
- `sample.geojson`
- `sample.kml`
- `sample.gpx`

For SHP ZIP testing, create a tiny shapefile in QGIS and zip `.shp`, `.shx`, `.dbf`, and `.prj` together. The app intentionally warns when `.prj` is missing because CRS may be unknown.
